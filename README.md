# Calorie Tracker / Gym Planner (PWA)


Full-stack Progressive Web App built with Laravel, JavaScript, MySQL, and Docker.

**Full source code available upon request.**

## Overview
Designed as a production-style application for tracking nutrition and planning training, with a minimalistic approach focused on core functionality for effective calorie tracking (weight loss/gain), built with a weightlifting use case in mind.

## Features
- Calorie tracking with full CRUD for food entries
- Gym planning and structured tracking
- Daily summaries (calories and macros)
- History view for past entries
- Food database search (nutrition lookup)
- Drag-and-drop meal organization
- Installable PWA (app-like experience)
